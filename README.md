## Description

---

- 🏗️개발 기간 : 2023.07 ~
- 🛠️사용기술
  - React, JavaScript, TypeScript, TailWind CSS
  - AWS S3

## 핵심 기능

- 사용자가 검색한 소환사의 게임 정보를 제공합니다.
- 게임 타입(랭크게임, 일반게임)을 설정하여 특정 게임 모드만 조회할 수 있습니다.
- 내가 자주 보고싶은 소환사를 즐겨찾기하여 더 편리하게 조회할 수 있습니다.
- 최근에 조회한 소환사의 이름을 까먹지 않을 수 있도록 최근 검색 기능을 제공합니다.
